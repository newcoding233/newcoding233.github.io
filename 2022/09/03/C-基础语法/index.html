    一、 数据类型

    不同数据类型所占的内存大小是取决于编译器和系统的位数，但相同编译器下不同系统之间基本的类型都是不变的，一直在变的只有指针所占的内存大小。

    1. 整型
      short: 占2byte 取值范围：(-2^15 ~ -2^15-1)
      int: 占4byte 取值范围：(-2^31 ~ -2^31-1)
      long: Windows为4byte；Linux为4字节(32位)，8byte(64位) 取值范围：(-2^31 ~ -2^31-1)
      long long：占8byte 取值范围：(-2^63 ~ -2^63-1)

    2. 浮点型
      float: 占4byte 有效数字范围：7位
      double: 占8byte 有效数字范围 15 ~ 16位

    3. 字符型和字符串类型
      char: 占1byte，字符类型变量不是把字符本身放到内存中储存，而是将对应的ASCII编码放入到存储单元。
      string:是C++在C的基础上重新定义的一个类，实质上是存储char*类型的容器，用于储存字符串。

    4. bool类型
      布尔类型只有true(1)和false(0)两个值，所以只占一个字节。

    5. sizeof 关键字
      sizeof(数据类型或者变量) 返回值为该数据类型或者变量所占的内存大小。
      cout << sizeof(int) << end; //输出int类型所占的内存大小

    三、运算符

    算术运算符

    ‘+’, ‘-‘, ‘*’, ‘/‘: 加减乘除
    ‘%’：取模，即两数相除，返回小数位的值
    ‘++’ 递增，’–’ 递减：’a++’为后置递增，即该行代码运算结束后对变量a做一个递增操作；’++a’前置递增，即该行代码运算开始之前对变量a做一个递增操作。

    赋值运算符

    比较运算符

    ‘==’相等，’!=’不等，’<’小于，’>’大于，’<=’小于等于，’>=’大于等于；比较运算符返回值都为0或者1。(0代表False，1代表True)

    逻辑运算符

    ‘!’ 非，取反的意思，真变假，假变真
    ‘&&’与，同真为真，否则为假
    ‘||’或，同假为假，否则为真

    四、选择、循环、跳转

    选择

    1. if 语句
      • 单行
        if(条件) 执行满足条件的语句
      • 多行
        if(条件) 执行语句A
        else 执行语句B
      • 多条件
        if(条件1) 执行语句1
        else if(条件2) 执行语句2
        else if(条件3) 执行语句3
        else 执行不满足上述条件的语句
        if语句内如果要执行的语句不止一行则需要使用{}来包含。推荐即使一句也使用{}，方便查阅。
    2. 三目运算符
      语句1 ? 语句2 : 语句3;
      语句1的返回值为bool类型
      当语句1的结果为真时，执行并返回语句2，反之返回语句3
    3. switch语句
      switch(表达式)
{
   case A:
     语句A;
     break;
   case B:
     语句A;
     break;
   ...
   default:
     执行语句;
     break;
}
      总结：switch语句只能在表达式的结果表示为一个具体的值时执行，不能在表示为一个区间时使用；而if语句是通用的；三目运算符简单高效，但条件限制过多，所以多用于核心代码编程。
