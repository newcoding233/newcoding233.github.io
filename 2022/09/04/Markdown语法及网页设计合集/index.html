    Markdown

    标题和正文

    • ‘#’为标题符号，一个为一级标题，两个为二级标题，以此类推(‘#’应该写在标题前)
    • 什么都不写就是正文

    分割线

    • 使用三个或以上的’-‘或者’*’表示

    字体

    • 斜体：’*内容*‘表示斜体
    • 粗体：’**内容**‘表示粗体
    • 删除线：’~~‘表示要删除线的文字

    超链接和图片

    • 超链接：’[连接体](地址)’
    • 图片：![图片体](地址)

    表格

    • 表格样式：
      |name|age|sex|
      |—|—|—|
      |张三|18|男|
      |李四|18|男|
      结果：
      name age sex
      张三 18 男
      李四 18 男
    • 对齐方式：
      ‘:-‘左对齐
      ‘:-:’居中
      ‘-:’右对齐
      |name|age|sex|
      |:—|:—: |—: |
      |张三|18|男|
      |李四|18|男|
      结果：
      name age sex
      张三 18 男
      李四 18 男

    代码块

    • 前后写上’```’表示代码块
      效果：
    int main()
{
    std::cout << "代码块" << std::endl;
}

    段落

    • 使用三个’空格’加回车
      效果如：
      最终，金岳霖没有等到林徽因，麦当没有找到彩虹海，曾小贤没有向胡一菲表白，陆展博没有等到林婉瑜……
      世间人，百般苦，百般丑，百般愁，百般情，亦是百般乐，百般美。
    • 使用空行
      效果如：
      最喜一袭素白，最喜千姿百态；最喜宁静深远，最喜闹市喧嚣；最喜家国天下，最喜家长里短。

    什么是诗？对生命中所有可能存在的美好的歌颂和赞美。对所有希冀和期望的畅想。

    [^要脚注的文本]

    效果：
    要脚注的文本^pos1。

    下划线

    <u>带下划线的内容</u>

    效果如：带下划线的内容

>区块引用的内容1

    高级链接

    使用变量来指代链接，再在文末附上变量地址，可以重复使用链接：
网站 Google1是搜索网站[Google1][pos1]
网站 Google2也是搜索网站[Google2][pos1]
Runoob是一个网站[Runoob][runoob]

  [1]: http://www.google.com/
  [runoob]: http://www.runoob.com/

    效果如下：
    网站 Google1是搜索网站Google1
    网站 Google2也是搜索网站Google2
    Runoob是一个网站Runoob

    流程图

    设置条件：名字=>类型: 描述
    常用的类型：

    1. 开始和结束：start、end
    2. 输入和输出：inputoutput
    3. 操作：opertion
    4. 条件：condition
    5. 子程序：subroutine
      代码：

      ```flow
      st=>start: 开始
      io=>inputoutput: 输入输出
      op=>operation: 你的操作
      cond=>condition: Yes or No ?
      sub=>subroutine: 你的子程序
      e=>end: 结束

      st->io->op->cond
      cond(Yes)->e
      cond(No)->sub->io
      ```

    效果如：

    st=>start: Start
io=>inputoutput: cath somthing
op=>operation: your operation
cond=>condition: Yes or No ?
sub=>subroutine: your subroutine
e=>end: end

st->io->op->cond
cond(yes)->e
cond(no)->sub->io

    更多请参考: http://flowchart.js.org/
